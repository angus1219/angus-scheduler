# Angus Scheduler

OE 课程排程系统 — 管理课程日期、场地、AV 系统。

## 功能

- 🗓️ 月历视图，一目了然
- 📋 课程 CRUD：新增、编辑、删除
- 🏠 支持自有场地（OE Ballroom, MDMT Ballroom）+ 外部场地
- 🎤 AV 系统分类（Inhouse / Outsource）
- 📱 PWA — 可添加到 iPhone 主屏幕，离线可用
- 💾 数据存于浏览器 IndexedDB，无需服务器

## 使用

打开 https://angus1219.github.io/angus-scheduler/

或添加到 iPhone 主屏幕：
1. Safari 打开
2. 分享按钮 → 「添加到主屏幕」
